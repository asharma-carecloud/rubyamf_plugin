rubyamf_plugin by victorcoder

Flash and Flex Remoting for Ruby on Rails

RubyAMF is an open source flash remoting gateway for Ruby on Rails. RubyAMF is a plugin for Ruby on Rails that plugs directly into your controllers with render :amf.

Install

From your rails application directory, run: ruby script/plugin install git://github.com/victorcoder/rubyamf_plugin.git
